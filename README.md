# Yii2 NMS - V.3.0

This branch contains development for Read , Edit , Delete features in the Network Monitoring System.

## Features

✅ IP and Device-type culomns added to db and form
✅ Display a list of devices
✅ Edit device information
✅ Delete devices
✅ Restrict edit/delete actions to the **admin** user
✅ Details option in status-monitoring
✅ Simple, lightweight design without complex plugins

## How to Use

1. Clone the project 

2. Configure your database connection in `config/db.php`.

3. Run migrations if any.

4. Open this link in your bowser http://yourLocalAddress/basic/web/index.php?r=site%2Findex 

5. Log in as user **admin** with password = 1 , to access edit and delete actions.
